whats binary classification

May 10
Binary classification means the output has only two possible answers — in your case:

0 = No heart disease
1 = Has heart disease
The model looks at the patient's vitals and decides which of these two categories they fall into. That's it — binary means two.

Compare it to:

Multi-class classification — more than 2 categories, like classifying a disease as mild, moderate, or severe
Regression — predicting a continuous number, like predicting exact cholesterol level
Your project is binary because the answer is simply yes or no — does this patient have heart disease or not.

learning curves is a model ?

May 15
No — learning curves are a visualization/evaluation technique, not a model.

It's a chart that shows:

X axis — how much training data you used
Y axis — accuracy score
It answers the question: "does my model get better as it sees more data?"

For example it shows you:

If training accuracy is high but test accuracy is low → model is overfitting (memorizing data, not learning)
If both are low → model is underfitting (not learning enough)
If both are high and close together → model is good
It's basically a diagnostic tool to understand how well your model is learning. One chart, very easy to add, and looks great in a presentation because you can explain what it means clinically.

Train — teach the model using the training data (734 patients). The model learns patterns from this data.

Evaluate — test the model on the test data (184 patients) that it has never seen before. This tells you how well it actually performs on new patients.
